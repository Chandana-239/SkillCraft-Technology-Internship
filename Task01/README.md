# Task 01 - Student Lifestyle Analysis Dashboard

## Objective
Analyze student lifestyle patterns and identify relationships between screen time usage and stress levels using interactive data visualizations.

## Tool Used
Tableau

## Dataset
Student Lifestyle Dataset

## Dashboard Highlights
* Distribution of Student Screen Time
* Distribution of Stress Levels
* Interactive visual analysis of student lifestyle factors

## Key Insights
* Most students reported medium stress levels.
* Screen time usage varied across students, with a significant number spending 7–10 hours on screens daily.
* The dashboard helps visualize lifestyle patterns and understand factors influencing student well-being.

## Outcome
Developed an interactive Tableau dashboard to analyze student lifestyle behaviors and stress patterns, enabling better understanding of student well-being through data visualization.
