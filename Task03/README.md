# Task 03 - Decision Tree Classifier using Bank Marketing Dataset

## Objective

Build a Decision Tree Classifier to predict whether a customer will subscribe to a term deposit based on demographic and marketing campaign information.

## Tools Used

* Python
* Pandas
* Scikit-Learn
* Matplotlib
* Seaborn

## Dataset

Bank Marketing Dataset

## Methodology

### Data Preprocessing

* Loaded the dataset using Pandas.
* Checked dataset structure and missing values.
* Converted categorical variables into numerical format using Label Encoding.

### Model Building

* Split the dataset into training and testing sets.
* Trained a Decision Tree Classifier using the entropy criterion.
* Used the trained model to predict customer subscription outcomes.

### Model Evaluation

* Calculated model accuracy.
* Generated a confusion matrix.
* Produced a classification report containing precision, recall, and F1-score.

### Visualization

* Visualized the Decision Tree structure.
* Identified the most important features influencing predictions using feature importance analysis.

## Outcome

Successfully developed a Decision Tree Classification model for customer subscription prediction and evaluated its performance using multiple metrics and visualizations.
